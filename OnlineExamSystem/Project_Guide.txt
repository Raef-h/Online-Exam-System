ONLINE EXAM SYSTEM - PROJECT DOCUMENTATION
==========================================

This guide explains how to set up, run, and verify the project.

--------------------------------------------------------------------------------
1. PREREQUISITES
--------------------------------------------------------------------------------
- Java JDK 21 or higher.
- XAMPP (for MySQL database).
- Project folder containing 'run.bat' and 'javafx-sdk'.

--------------------------------------------------------------------------------
2. DATABASE SETUP (XAMPP)
--------------------------------------------------------------------------------
1. Open XAMPP Control Panel.
2. Click "Start" next to MySQL.
3. The application will handle database and table creation automatically.

[SCREENSHOT TIP: Take a screenshot of the XAMPP Control Panel with MySQL running]

To view the results in the database:
1. Open browser and go to: http://localhost/phpmyadmin/
2. Select the "onlineexam" database.
3. Open the "ExamResults" table.

[SCREENSHOT TIP: Take a screenshot of the phpMyAdmin interface showing the table structure]

--------------------------------------------------------------------------------
3. RUNNING THE APPLICATION
--------------------------------------------------------------------------------
1. Go to the project folder.
2. Double-click the file "run.bat".
3. A terminal will open to compile and launch the application.

[SCREENSHOT TIP: Take a screenshot of the terminal showing "[OK] Build successful!"]

--------------------------------------------------------------------------------
4. SERVER (DOCTOR) WORKFLOW
--------------------------------------------------------------------------------
1. On the "mode_selection.fxml" window, click "Start Exam Server".
2. The Server window opens. Go to "File" -> "New Exam".
3. Fill in:
   - Course Name (e.g., CPCS-405)
   - Year (e.g., 2024)
   - Semester (e.g., First)
4. Click "Import File" and choose "Sample_Questions_Final.txt".
5. Click "Create Exam".

[SCREENSHOT TIP: Take a screenshot of the "New Exam.fxml" window after filling data]
[SCREENSHOT TIP: Take a screenshot of the main Server window showing the active exam in the table]

--------------------------------------------------------------------------------
5. CLIENT (STUDENT) WORKFLOW
--------------------------------------------------------------------------------
1. Run "run.bat" again and click "Join Exam".
2. Enter:
   - Student ID: (Any number)
   - Server IP: 127.0.0.1
   - Port: 3000
3. Select the exam from the dropdown list.
4. Answer the questions and finish the exam.

[SCREENSHOT TIP: Take a screenshot of the Login window]
[SCREENSHOT TIP: Take a screenshot of a question during the exam]
[SCREENSHOT TIP: Take a screenshot of the "Exam Complete" success dialog]

--------------------------------------------------------------------------------
6. FINAL VERIFICATION
--------------------------------------------------------------------------------
1. Check the Server window's log at the bottom to see the student's score.
2. Check the "exam_log.txt" file in the project folder.
3. Refresh phpMyAdmin to see the new record in the "ExamResults" table.

[SCREENSHOT TIP: Take a screenshot of the final record in the MySQL table]
